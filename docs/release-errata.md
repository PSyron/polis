# Release errata

This file is append-only. Corrections clarify published evidence without moving a

## 0.1.0 documentation drift (2026-07-22)

After `v0.1.0` was published, two later LanguageTool changes were accidentally added
to the `0.1.0` changelog section and release-notes file. Those files have been
restored byte-for-byte to the content in `v0.1.0`. The later changes are recorded
under `Unreleased` in `CHANGELOG.md`; no `v0.1.0` tag or release asset was changed.
